# Actividad #2 - Página Web Interactiva

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de una página web interactiva utilizando HTML, CSS y JavaScript. Se aplican conceptos básicos de estructura, diseño y funcionalidad en el navegador.

El trabajo se realiza en equipo utilizando GitHub como herramienta de control de versiones.

---

## Tecnologías Utilizadas
- HTML
- CSS
- JavaScript
- Git
- GitHub

proyecto-web/
│── index.html
│── styles.css
│── script.js

## Organización del Equipo
- Líder del proyecto: Encargado del repositorio y del encabezado (header)
- Colaborador 1: Desarrollo del contenido principal (body)
- Colaborador 2: Desarrollo del pie de página (footer)

---

## Flujo de Trabajo

### 1. Creación del repositorio
- Creación del repositorio en GitHub
- Inicialización con README.md
- Adición de colaboradores

### 2. Estructura inicial
- Creación de archivos base
- Primer commit en la rama main

### 3. Trabajo con ramas
Cada integrante trabaja en su propia rama:
- feature/ENCABEZADO
- feature/CUERPO
- feature/PIE-PAGINA

### 4. Desarrollo
Cada integrante desarrolla su parte correspondiente del proyecto.

### 5. Commits
Se realizan commits claros y específicos describiendo los cambios realizados.

Ejemplos:
- Se agrego estructura base
- Se implemento menu de navegacion
- Se agrego funcionalidad con JavaScript

### 6. Pull Requests
- Cada integrante sube su rama
- Se crea un Pull Request hacia main
- El líder revisa y aprueba los cambios

### 7. Integración final
- Revisión completa del proyecto
- Resolución de conflictos si existen

### 8. Publicación
El proyecto se publica utilizando GitHub Pages desde la rama main.
